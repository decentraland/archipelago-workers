Archipelago Stats API (1.0.0)

Download OpenAPI specification:

URL: https://decentraland.org

API for archipelago statistics service providing parcel, peer, and island data

Get parcels with peer counts

Returns list of parcels with their peer counts

Responses

Service root

/parcels

Response samples

  • 200
Content type
application/json
[
  • {
    • "peersCount": 0,
    • "parcel": {
      • "x": 0,
      • "y": 0
      }
    }
]
